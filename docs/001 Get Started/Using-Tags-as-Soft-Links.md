# Tags and their purpose
Tags are a tool to provide loose cross-structure to an other wise rigid existing hierarchy. 

# Tags used in this Digital Garden

| Tag Class                   | Tag                  | Tag Description                                                                                                                                                                                 |
| --------------------------- | -------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| System                      | #Dashboard      | Deals with notes that consolidate content in a single place. Most of these links are generated by queries or implicit links.                                                                    |
| System                      | #MOC            | MOC or Maps of Content aid in structuring related notes together for note-surfing serendipity. Links here are mostly made explicitly.                                                           |
|                             |                      |                                                                                                                                                                                                 |
| Note Taking and Note Making | #Fleeting       | Notes are made to quickly jot down words and ideas. They usually are immature and would eventually be converted to evergreen notes. A note must not be perpetually a fleeting note.             |
| Note Taking and Note Making | #Reference      | Notes taken from any available source in the world. This mostly sticks to its source material. Proper citations are mandatorily required.                                                       |
| Note Taking and Note Making | #Evergreen      | Notes that are from the author's own words and understanding. This can arise out of sheer thought or the combinational creativity from multiple references.                                     |
| Note Taking and Note Making | #Journal        | Notes that are journal entries. They capture growth, memories, mistakes and mitigations.                                                                                                        |
|                             |                      |                                                                                                                                                                                                 |
| Content Consumption         | #Curriculum     | Notes that arise in direct reference to a tutorial or course or a curriculum. This can be of any medium like traditional - schools, university or modern - online courses, bootcamps and MOOCs. |
| Content Consumption         | #Literature     | Notes that arise in direct reference to content in a book, research paper or any other literary work.                                                                                           |
| Content Consumption         | #Articles       | Notes that arise in direct reference to content in articles and blogs of physical or digital format.                                                                                            |
| Content Consumption         | #Audios         | Notes that arise in direct reference to content in a podcast or any other audio source.                                                                                                         |
| Content Consumption         | #Music          | Notes that arise in direct reference to content in a musical piece or its performance.                                                                                                          |
| Content Consumption         | #Videos         | Notes that arise in direct reference to content in a video format of both informational and entertainment in nature.                                                                            |
|                             |                      |                                                                                                                                                                                                 |
| Entities                    | #People         | Notes that refer to people - living, dead, fictional and anything in-between.                                                                                                                   |
| Entities                    | #Establishments | Notes that refer any company, organization, governments or any other autonomous body of a collection of like-minded beings.                                                                     |
| Entities                    | #Places         | Notes that refer places of both real and fictional in nature.                                                                                                                                   |
