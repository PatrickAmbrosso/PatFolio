# The fundamentals of programming
This particular guide (of sorts) is extracted with heavy inspiration from the video by [FreeCodeCamp](https://www.freecodecamp.org/) by the name [Introduction to Programming and Computer Science](https://www.youtube.com/watch?v=zOjov-2OZ0E) and it deals with the most common aspects of programming in general while staying language and technology agnostic. This is accomplished by discussing the fundamental concepts that exist across all programming languages.

By the end of this guide, one will be equipped enough to answer the following questions confidently.
1. What is programming?
2. How to write code?
3. How to get information from computers?
4. What can computers do?
5. What are variables
6. How to manipulate variables?
7. What are conditional statements?
8. What are arrays?
9. What are Loops?
10. What are errors?
11. How to debug code?
12. What are functions?
13. How to import functions?
14. How to create functions?
15. What are array lists and dictionaries?
16. What are data structures?
17. What is recursion?
18. What is pseudocode?
19. Choosing the right programming language
20. Applications of programming
